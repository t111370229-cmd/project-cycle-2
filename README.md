## Project Overview

- Dataset used: `YRBS_2007.csv`
- Selected proportion variable: `CurrentAlcoholUse` with benchmark `p0 = 0.35`
- Selected mean variable: `HowTallAreYouWithoutShoesInMeters` with benchmark `mu0 = 1.70`
- Proportion research question: Is the proportion of students who reported any current alcohol use different from 0.35?
- Mean research question: Is the mean self-reported height without shoes different from 1.70 meters?

## Final Conclusion

The estimated proportion of students with current alcohol use was 45.17%
(95% CI: 44.31% to 46.04%), so the one-sample proportion
test rejected `H0: p = 0.35`. The sample mean height was 1.694 m
(95% CI: 1.692 to 1.696), so the one-sample t-test
rejected `H0: mu = 1.70`. The alcohol-use result is substantively large, while the
height result is statistically significant but practically small.
